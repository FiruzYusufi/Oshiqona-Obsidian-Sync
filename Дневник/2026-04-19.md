## Заметки и идеи

> Проект Oshiqona — Angular 20 + Nx monorepo. Хорошая архитектура с разделением на libs.
> SignalR для реал-тайм — правильный выбор для .NET backend.
> Taiga UI — мощная библиотека компонентов.

## Архитектурные заметки

```
apps/admin  → @core, @core/auth, @shared/cdk, @shared/kit
apps/client → @core, @core/auth, @shared/cdk, @shared/kit
```

Все библиотеки независимы, что позволяет легко тестировать и переиспользовать.
