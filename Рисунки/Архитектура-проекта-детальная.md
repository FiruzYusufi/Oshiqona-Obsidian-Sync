## 🔄 Потоки данных и взаимодействия

### 🔐 Поток аутентификации
```
1. User Login Request
   ↓
2. AuthService.login()
   ↓
3. HTTP POST /api/auth/login
   ↓
4. JWT Token Response
   ↓
5. TokenService.setToken()
   ↓
6. CurrentUserService.setUser()
   ↓
7. Router Navigation
   ↓
8. authGuard/clientAuthGuard Check
```

### 💬 Поток реал-тайм чата
```
1. User Opens Chat
   ↓
2. ChatHubService.connect()
   ↓
3. SignalR Connection Established
   ↓
4. ChatHubService.joinChat(chatId)
   ↓
5. User Types Message
   ↓
6. MessageService.sendMessage()
   ↓
7. ChatHub.SendMessage()
   ↓
8. Real-time Message Broadcast
   ↓
9. Other Users Receive Message
   ↓
10. UI Updates Automatically
```

### 🎯 Поток системы рекомендаций
```
1. User Opens Matches Page
   ↓
2. RecommendationService.getRecommendations()
   ↓
3. HTTP GET /api/recommendations
   ↓
4. Algorithm Calculates Matches
   ↓
5. User Cards Displayed
   ↓
6. User Swipes (Like/Dislike)
   ↓
7. SwipeService.likeUser()/dislikeUser()
   ↓
8. HTTP POST /api/matches/like
   ↓
9. Check for Mutual Match
   ↓
10. Match Notification (if matched)
```
