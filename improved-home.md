## 🎨 Приложения

### 👨‍💼 Admin App (порт 4000)
- **Dashboard** — метрики и мониторинг
- **References** — справочники системы
- **User Management** — управление пользователями
- **Guard**: `authGuard` + группа `Administrators`

### 👥 Client App (порт 4001) 
- **Matches** — система рекомендаций
- **Chats** — реал-тайм сообщения
- **Profile** — профили пользователей
- **Gallery** — управление фотографиями
- **Guard**: `clientAuthGuard`

## 📋 Быстрые команды

### Разработка
```bash
npm run start:oshiqona-client   # Client (4001)
npm run start:oshiqona-admin    # Admin (4000)
npm run build:prod              # Production build
npm run test                    # Тесты
npm run lint                    # Линтер
```

### Obsidian
- `Ctrl+O` — быстрый переход к файлу
- `Ctrl+Shift+F` — глобальный поиск
- `Ctrl+G` — граф связей
- `Ctrl+P` — палитра команд
