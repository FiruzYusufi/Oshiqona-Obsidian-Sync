# Команды разработки

## Запуск приложений

```bash
# Запустить оба приложения
npm run start

# Только Admin (порт 4000)
npm run start:oshiqona-admin
npx nx serve oshiqona-admin

# Только Client (порт 4001)
npm run start:oshiqona-client
npx nx serve oshiqona-chat
```

## Сборка

```bash
# Production build (оба приложения)
npm run build:prod
npx nx run-many -t build -c production

# Build конкретного приложения
npx nx build oshiqona-admin -c production
npx nx build oshiqona-chat -c production
```

## Тестирование

```bash
# Все тесты
npm run test
npx nx run-many -t test

# Тесты конкретного проекта
npx nx test oshiqona-admin
npx nx test oshiqona-chat
npx nx test core
npx nx test core-auth

# С покрытием
npx nx test core --coverage

# Watch режим
npx nx test core --watch
```

## Линтинг

```bash
# Все проекты
npm run lint
npx nx run-many -t lint

# Конкретный проект
npx nx lint oshiqona-admin
npx nx lint oshiqona-chat

# Автоисправление
npx nx lint oshiqona-admin --fix
```

## Проверка типов

```bash
# Строгая проверка TypeScript
npm run typecheck
tsc --noEmit --strict --project tsconfig.typecheck.json
```

## Генерация кода

```bash
# Новый компонент в Admin
npx nx g @nx/angular:component pages/my-page --project=oshiqona-admin

# Новый компонент в Client
npx nx g @nx/angular:component pages/my-page --project=oshiqona-chat

# Новый компонент в shared/kit
npx nx g @nx/angular:component components/my-comp --project=shared-kit

# Новая библиотека
npx nx g @nx/angular:library my-lib --directory=libs/my-lib

# Новый сервис
npx nx g @nx/angular:service services/my-service --project=core

# Новая директива
npx nx g @nx/angular:directive directives/my-directive --project=shared-cdk

# Новый пайп
npx nx g @nx/angular:pipe pipes/my-pipe --project=shared-cdk
```

## Nx утилиты

```bash
# Граф зависимостей
npx nx graph

# Список проектов
npx nx show projects

# Информация о проекте
npx nx show project oshiqona-chat

# Очистить кэш
npx nx reset

# Обновить Nx
npx nx migrate latest
```

## Git команды

```bash
# Стандартный коммит (commitlint)
git commit -m "feat(chat): add real-time typing indicator"
git commit -m "fix(auth): fix token refresh on 401"
git commit -m "docs(readme): update setup instructions"
git commit -m "refactor(profile): extract photo upload logic"
git commit -m "test(chat): add unit tests for ChatService"
git commit -m "style(layout): fix mobile navigation spacing"
git commit -m "chore(deps): update Angular to 20.3.15"

# Типы коммитов
# feat     — новая функция
# fix      — исправление бага
# docs     — документация
# style    — форматирование
# refactor — рефакторинг
# test     — тесты
# chore    — обслуживание
# perf     — производительность
# ci       — CI/CD
```

## Docker

```bash
# Собрать образ
docker build -t oshiqona:latest .

# Запустить контейнер
docker run -p 3000:3000 -p 3001:3001 oshiqona:latest

# Docker Compose
docker-compose up -d
docker-compose down
docker-compose logs -f
```
