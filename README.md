## 🔄 Git синхронизация

Автоматическая синхронизация каждые 10 минут:
- **Auto commit** — сохранение изменений
- **Auto push** — отправка в репозиторий
- **Auto pull** — получение обновлений

### Ручная синхронизация
- `Ctrl+Shift+G` — открыть Git панель
- `Ctrl+P` → "Git: Commit all changes"
- `Ctrl+P` → "Git: Push"
- `Ctrl+P` → "Git: Pull"

## 📋 Быстрые команды

### Навигация
- `Ctrl+O` — быстрый переход к файлу
- `Ctrl+Shift+F` — глобальный поиск
- `Ctrl+G` — граф связей
- `Ctrl+P` — палитра команд

### Создание контента
- `Ctrl+N` — новая заметка (выбрать шаблон)
- `Alt+E` — новая Excalidraw диаграмма
- `Ctrl+Shift+K` — новая Kanban доска

### Kanban
- Перетаскивание карточек между колонками
- `#critical` `#bug` `#feature` — теги для фильтрации
- Цветовая кодировка по приоритету
