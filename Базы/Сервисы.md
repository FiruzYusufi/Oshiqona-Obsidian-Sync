## API Endpoints

| Endpoint | Метод | Сервис | Описание |
|----------|-------|--------|---------|
| `/api/auth/sign-in` | POST | AuthService | Вход |
| `/api/auth/sign-up` | POST | AuthService | Регистрация |
| `/api/auth/sign-out` | POST | AuthService | Выход |
| `/api/auth/refresh-token` | POST | TokenService | Обновление токена |
| `/api/users/me` | GET | CurrentUserService | Текущий пользователь |
| `/api/users/me` | PUT | CurrentUserService | Обновить профиль |
| `/api/users/me/photos` | POST | CurrentUserService | Загрузить фото |
| `/api/users` | GET | UserService | Список пользователей |
| `/api/users/:id` | GET | UserService | Пользователь по ID |
| `/api/chats` | GET | ChatService | Список чатов |
| `/api/chats` | POST | ChatService | Создать чат |
| `/api/chats/:id/messages` | GET | MessageService | Сообщения чата |
| `/api/chats/:id/messages` | POST | MessageService | Отправить сообщение |
| `/api/recommendations` | GET | RecommendationService | Рекомендации |
| `/api/swipes/like` | POST | SwipeService | Лайк |
| `/api/swipes/dislike` | POST | SwipeService | Дизлайк |
| `/api/swipes/matches` | GET | SwipeService | Совпадения |
