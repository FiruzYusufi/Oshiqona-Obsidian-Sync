# База данных — Компоненты

```dataview
TABLE
  file.folder AS "Папка",
  tags AS "Теги",
  file.mtime AS "Обновлено"
FROM "Проекты/Oshiqona/Документация"
WHERE contains(tags, "компоненты")
SORT file.name ASC
```
